# SkinTrack

## Project Description

SkinTrack is a web app designed to help users organize and track their skincare routines in one place. The app allows users to manage morning and evening routines, keep track of products, and stay consistent with their daily skincare habits through a simple and organized interface.

## Planned Features

1. Add skincare products to a routine  
2. Create separate morning and night routines  
3. Mark routine steps as completed  
4. Delete products or steps from the routine  
5. Responsive design for mobile and desktop users  

## Feature Added (Authentication)

For this assignment, I added authentication to the application. Users can create an account, log in, log out, and access the skincare tracker only when authenticated.

## What the Feature Does

- Users can sign up with an email and password  
- Users can log in and log out  
- The skincare tracker is protected and only visible when logged in  
- Each user’s skincare routines are saved separately  

## Service Used

This project uses **JavaScript localStorage** to simulate authentication and store user data. It also stores each user's skincare routines separately based on their login.

## How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in a web browser  
   OR use Live Server in VS Code  
4. Create an account and log in to use the app
