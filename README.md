# SkinTrack

## Project Description

SkinTrack is a web app designed to help users organize and track their skincare routines in one place. The app will allow users to manage morning and evening routines, keep track of products, and stay consistent with their daily skincare habits through a simple and organized interface.

## Planned Features

1. Add skincare products to a routine
2. Create separate morning and night routines
3. Mark routine steps as completed
4. Delete products or steps from the routine
5. Responsive design for mobile and desktop users
